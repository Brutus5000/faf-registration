On double-clicking the installer you will encounter a warning from Windows, because our installer is not signed. Signing requires a costly certificate, therefore we don't do it.
  1. Extract the .tar.gz file somewhere into your home directory.
  2. Install a Java JRE or JDK in version 14 (we recommend AdoptOpenJDK).
  3. Setup Proton in Steam to run Forged Alliance
    1. In the preferences enable Steam -> Settings -> Steam Play -> under Advanced -> "Enable Steam Play for all other titles"
    2. Set the launch options to: PROTON_NO_ESYNC=1 PROTON_DUMP_DEBUG_COMMANDS=1 %command%
    3. The best known working Proton version as of now is 5.0-9
    4. Run the game at least once. This will create a launcher shell script in /tmp/proton_your_linux_username
    5. If you ran the game from Steam you should have a file called "run" in /tmp/proton_your_linux_username.
    6. The tmp directory is only temporary so copy this file to somewhere nice. You will need it in the configure step.

The full and updated guide is in the wiki. We recommend the installation using native Steam.
